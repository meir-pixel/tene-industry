## בעיה A — כפתורי ערוצים מיותרים

**היום:** וואטסאפ / מייל / תמונה / טלפון מופיעים ככפתורים גדולים בראש תור האישור — נראים כמו פעולות, אבל הם רק פילטר.

**הטעות:** ערוץ המקור הוא **מטא-דאטה של ההזמנה**, לא פעולה. ההזמנה כבר נכנסה דרך הערוץ — אין מה ללחוץ.

**התיקון:**
1. **ערוץ = תגית קטנה בשורת ההזמנה** — `📱 וואטסאפ` / `📧 מייל` / `📷 תמונה` / `☎️ טלפון`. צבע עדין, לא כפתור.
2. **פילטר = dropdown קטן אחד** ("סנן לפי מקור ▾") — לא 5 כפתורים. ברירת מחדל: הכל.
3. הסרת הכפתורים הגדולים לגמרי.

```
לפני:  [הכל] [וואטסאפ] [מייל] [תמונה] [טלפון]   ← 5 כפתורים ענקיים
אחרי:  סנן: [הכל ▾]                              ← dropdown קטן
       + בכל שורה: תגית "📱 וואטסאפ"
```

## בעיה B — מסך השוואת OCR גרוע

**היום:** השוואה בין התמונה המקורית למה שה-AI זיהה — לא נוחה, קשה להשוות.

**התיקון — צד-ליד-צד עם תיקון במקום:**
```
┌────────────────────┬────────────────────┐
│   מקור (תמונה)      │   זיהוי (ניתן לעריכה)│
│                     │                     │
│  [תמונת ההזמנה]     │  קוטר: [12    ] ✏️  │
│   זום / החלקה       │  כמות: [50    ] ✏️  │
│                     │  צלעות: [...]  ✏️   │
│                     │  ⚠️ שדה לא בטוח     │
└────────────────────┴────────────────────┘
        [ אשר ויצור הזמנה ]   [ דחה ]
```

**עקרונות:**
1. **תמונה משמאל, נתונים מימין** — תמיד רואים את שניהם יחד.
2. **תיקון באותו מסך** — לוחצים על שדה שגוי ומתקנים במקום (לא מסך נפרד). ← אושר ע"י מאיר.
3. **סימון ביטחון** — שדה שה-AI לא בטוח בו (confidence נמוך) מודגש בצהוב, כדי שהעין תלך לשם.
4. **זום בתמונה** — אפשר להגדיל אזור בתמונה כדי לקרוא כתב יד.
5. **כל תיקון נשמר** — מועבר אוטומטית למודול האימון (בעיה C) ללימוד עתידי.

---

## בעיה C — אימון OCR לא שייך לקליטה

**היום:** "אימון מודולים / OCR" תחוב בתוך מסך קליטת הזמנות — לא המקום שלו. פקיד שקולט הזמנה לא צריך לראות כלי תחזוקה של ה-AI.

**התיקון — מודול נפרד:**
- מסך עצמאי **"אימון זיהוי"** (`ocr-training.html`), נגיש מ**הגדרות / ניהול**, לא מהקליטה.
- מה הוא מציג: רשימת התיקונים שנעשו (מה ה-AI זיהה ↔ מה תוקן), כדי לשפר prompts/דוגמאות.
- כל תיקון ממסך ההשוואה (בעיה B) זורם לכאן אוטומטית.
- מי ניגש: manager/admin בלבד.

```
קליטת הזמנות (יומיומי, פקיד)     אימון זיהוי (תחזוקה, מנהל)
─────────────────────────         ──────────────────────
תור אישור + השוואת OCR            רשימת תיקונים → שיפור ה-AI
```

הסרת בלוק "אימון / OCR" ממסך הקליטה לחלוטין.
